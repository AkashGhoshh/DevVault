# Application Goal
This is a **Production-Level Application** (Dev Vault).

## Rules for Agents
1. **No Hacky Code:** All features must be built cleanly, optimized, and ready for end-user production usage.
2. **Data Safety First:** Never write code that could accidentally lose user notes or data. Always validate storage processes.
3. **Flexible & Secure Architectures:** When adding third-party integrations (like AI), always use standard, flexible approaches (e.g., standard OpenAI-compatible endpoints) rather than hardcoding vendor-specific implementations, giving the end user maximum privacy and control.
